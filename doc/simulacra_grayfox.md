# グレイフォックス

## アバター特性

* 最終ダメージ+18%
* <mark>恩恵共鳴発動中、魔力ゲージが最大スタックの時、スキル-魔力変換を発動すると、自身と付近にいる仲間の最終ダメージ+10%</mark>
* 重複不可、30秒持続

## ボリション

### 2セット
* 恩恵共鳴以外を発動している場合、氷攻撃力が24%/26%/28%/30%増加
* <mark>恩恵共鳴を発動している場合、全属性攻撃力が24%/26%/28%/30%増加</mark>
* 回復効果が20%増加
* 戦闘開始時に1,000ポイントのエナジーを即時獲得、15秒間に1回のみ発動可能
* バックグラウンド、最高ランクのみが有効

### 4セット
* 連携スキル-タイムスパークを発動すると、最終ダメージが22%/26%/30%/34%増加
* 恩恵共鳴以外の時、追加で氷ダメージ+10%増加、40秒持続
* <mark>恩恵共鳴を発動している場合、自身と付近の仲間の会心率が20%、会心ダメージが12%増加、40秒持続</mark>
* バックグラウンド、最高ランクのみが有効

```
恩恵：4セット必須、無凸でもOK
```

## 連携スキル：タイムスパーク
* ただちに魔力ゲージを12と、30秒間不屈効果を獲得
* また周囲の敵に攻撃力x4500%のダメージを与える
* スキル発動中、被ダメージ無効が1.5秒間持続
* <mark>恩恵共鳴を発動している場合、自身と周囲の仲間のステータス異常と行動制限と灰蝕を解除</mark>

## スキル：魔法変換
* 魔力ゲージのスタックを全て消費し、付近のターゲットに攻撃力x2093%のダメージを与える
* 「時の魔女」状態に入ると一部の通常攻撃が消費した魔力ゲージのスタック数に基づいて変化する
* 1スタック消費するごとに、氷ダメージ+1%、30秒間持続
* 効果持続中に通常攻撃-クロノスタシスを発動すると、新しいスキルフリーズビームを獲得できる
* スキルフリーズビームは、時の魔女の発動1回につき、1回のみ獲得/発動できる、武器を切り替えると獲得/発動できなくなる
* クールダウン30秒、リセット不可

### フリーズビーム
* 時の魔女状態のときに発動できる
* 付近のターゲットに攻撃力x1261%のダメージを与え、2秒間凍結させる
* <mark>付近にいる仲間のHPを攻撃力x400%HPを回復し、魔力ゲージを12スタック獲得</mark>

## 通常攻撃
### 魔法解放
* 地上で通常攻撃を使用すると最大4回まで攻撃を行える
* 1回目：対象に対して攻撃力x270%のダメージを与え、ノックバックし、魔力ゲージを1スタック獲得
* 2回目：対象に対して攻撃力x374%のダメージを与え、ノックバックし、魔力ゲージを1スタック獲得
* 3回目：対象に対して攻撃力x386%のダメージを与え、ノックバックし、魔力ゲージを2スタック獲得
* 4回目：対象に対して攻撃力x594%のダメージを与え、ノックバックし、魔力ゲージを2スタック獲得
* 通常攻撃-魔力解放が命中するごとに、ターゲットにタイムトリックを1スタック付与する
* タイムトリックはターゲットの全属性の攻撃力を3%ダウンさせる
* 10秒間持続、最大4スタックまで可能

### 翔空魔舞
* 空中で通常攻撃を使用すると最大4回まで攻撃を与える
* 1回目：対象に対して1078%の攻撃力+4762のダメージを与え、ターゲットをノックバックし、威能ゲージを1層獲得する
* 2回目：対象に対して1494%の攻撃力+6600のダメージを与え、ターゲットをノックバックし、威能ゲージを1層獲得する
* 3回目：対象に対して1543%の攻撃力+6817のダメージを与え、ターゲットをノックバックし、威能ゲージを2層獲得する
* 4回目：対象に対して2376%の攻撃力+10479のダメージを与え、ターゲットをノックバックし、威能ゲージを2層獲得する
* 攻撃が命中したターゲットに対し「時機錯誤」効果を付与し、全攻撃力を3%現象させる
* この効果は10秒間持続、同一対象に最大4層まで重複可能

### 棘輪炸裂
* 地上で通常攻撃を行っているとき、3回目から4回目の通常攻撃中に長押しで発動
* 周囲の出来に4000%の攻撃力+17672のダメージを与え、ターゲットをノックバックさせる
* 攻撃が命中したターゲットに対し「時機錯誤」効果が一時的に全攻撃力を5%低下させる
* さらに対象に10%の減速効果を付与
* この効果は10秒間持続

# 恩恵目線の考察

## ガチャ目標

* 武器は5凸
* ボリションは無凸4セット
* アバター特性はコンテンツ次第なので任意(恩恵意外は使う必要なし)

## 概要

* 連携スキル：威能ゲージ+12層、不屈、自身と仲間の会心率+16%(ボリ4)、自身と仲間の会心ダメ+7%(ボリ4)、デバフ解除、灰蝕解除
* スキル：威能ゲージ全消費、時の魔女、最終ダメージ+25%(3凸)、最終ダメージ+5%(ボリ2)、即死回避付与(5凸)
* 通常攻撃：威能ゲージを12層貯める
* 通常攻撃長押し：威能ゲージ-12層、デバフ解除、灰蝕解除、過去3.5秒間に失ったHP全回復+攻撃力*200%回復
* スキル2：威能ゲージ+12層、攻撃力*400%回復

## 特徴

* 戦闘の状況、他武器／ボリションの組み合わせを考慮して瞬間的に判断して仕様する必要あり、恩恵武器としては比較的使うのが難しい部類
* 基本的にグレイフォックスを表にして敵に攻撃しながら使う武器
* 近接戦闘が要求されるためある程度の耐性が必要
* 持続回復なし
* 連携スキルは発動必須だがキャンセルも必須（時間がもったいない）
* スキルも必須、これはキャンセルしない
* ブレヴィ／ゼロとの相性は最悪

## 基本編成

| 武器 | ボリション |
| --- | --- |
| グレイフォックス(魔女の鍵) | グレイフォックスx4 |
| エスター(夕べの祈り) | フィオナx4 |
| フィオナ(月星の環) | ブレヴィx4 |

* 基本になる構成、この構成だとフィオナを使うことはない
* フィオナの第2スキルはトレンドフォースにすると回しやすくなる
* 通常攻撃で魔力ゲージを貯める際のテクニックとして、通常攻撃2段目でジャンプキャンセルすると最速で魔力ゲージが貯められる
  * 通常1段目 → 通常2段目 → すぐにジャンプ(タイミングがシビア) → 通常1段目 ...(繰り返し)
  * 通常3段目でキャンセルする方法もあるが、敵から少し離れてしまうデメリットがある

1. エスター：切っ先を3つ設置
2. グレイフォックス：連携スキル(キャンセル) ... 魔力ゲージx12が溜まる、不屈獲得、会心率+20%、会心ダメージ+12%
3. グレイフォックス：スキル(打ち切る) ... 魔力ゲージ全消費、時の魔女状態になる
4. グレイフォックス：通常攻撃 ... 魔力ゲージx12を貯める
5. グレイフォックス：通常長押し ... 回復
6. 4～5の繰り返し
6. グレイフォックス：第2スキル(グレイフォックスのスキルCDが3秒程度になったら)
8. 1に戻る

  * グレイフォックスとゼロの相性が最悪
