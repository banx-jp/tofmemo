# グレイフォックス

## アバター特性

* 最終ダメージが18%増加
* <mark>恩恵共鳴を発動すると威能ゲージを消費してスキル「魔力変換」を発動後、自身とチームメンバーに5%の最終ダメージ増加効果を獲得</mark>
* 重複不可、30秒持続

## ボリション

### 2セット
* 回復効果が20%増加
* 戦闘開始時に1,000ポイントのエナジーを即時獲得、15秒間に1回のみ発動可能
* <mark>恩恵共鳴を発動している場合、全属性攻撃力が24%/26%/28%/30%増加</mark>
* 恩恵共鳴以外を発動している場合、氷攻撃力が24%/26%/28%/30%増加
* バックグラウンド、最高ランクのみが有効

### 4セット
* 連携スキル「時空花火」を発動すると、最終ダメージが22%/26%/30%/34%増加
* <mark>連携スキル「時空花火」を発動させた時、恩恵共鳴を発動している場合、自身と近くの仲間の会心率が16%、会心ダメージが7%増加、40秒持続</mark>
* 連携スキル「時空花火」を発動させた時、恩恵共鳴以外が発動している場合、氷ダメージがさらに10%増加、40秒持続
* バックグラウンド、最高ランクのみが有効

```
恩恵：4セット必須、無凸でもOK
```

## 連携スキル：時空花火
* ただちに魔力ゲージを12と、30秒間不屈効果を獲得
* また周囲の敵に攻撃力x4500%のダメージを与える
* スキル発動中、被ダメージ無効が1.5秒間持続
* <mark>恩恵共鳴を発動している場合、自身と周囲の仲間のステータス異常と行動制限と灰蝕を解除</mark>

## スキル：魔法変換
* 魔力ゲージのスタックを全て消費し、付近のターゲットに攻撃力x2093%のダメージを与える
* 「時の魔女」状態に入ると一部の通常攻撃が消費した魔力ゲージのスタック数に基づいて変化する
* 1スタック消費するごとに、氷ダメージ+1%、30秒間持続
* 効果持続中に通常攻撃-クロノスタシスを発動すると、新しいスキルフリーズビームを獲得できる
* スキルフリーズビームは、時の魔女の発動1回につき、1回のみ獲得/発動できる、武器を切り替えると獲得/発動できなくなる
* クールダウン30秒、リセット不可

### 零点光束
* 「時の魔女」状態で使用できる
* 周囲のターゲットに1261%の攻撃力+22279のダメージを与える
* <mark>周囲の味方に攻撃力*400%に相当するHPを回復させ、12層の威能ゲージを獲得</mark>

## ランクアップ

### ★1
* <mark>「時の魔女」状態で毎秒1層の威能ゲージを追加で獲得</mark>
* スキル「魔力変換」を発動すると、命中した対象に追加「逆流の刻」効果を付与し、12秒間持続
* 毎秒1層の逆流効果を回収し、逆流の刻を持つ敵の数に応じて自身のHPを攻撃力*80%回復
* この効果は重複しない

### ★3
* <mark>恩恵共鳴を発動し、威能ゲージを12層保持している状態でスキル「魔力変換」を発動すると自身と仲間の最終ダメージが25%増加、30秒持続</mark>
* この効果は重複しない
* 「時の魔女」状態で毎秒、周囲の広範囲の敵に対して攻撃力*2400%のダメージを与える
* 「逆流の刻」効果が付与された敵は、2秒ごとに会心抵抗が5%減少し、10秒持続、この効果は最大5層まで

### ★5
* <mark>自身の回復効果が15%増加</mark>
* <mark>威能ゲージを12層消費するごとに、「免疫権限」を1層獲得し、致命的なダメージを1度だけ防ぐ</mark>
* <mark>致命的なダメージを防いだあと、「免疫権限」を1層消費し、最大HPの15%を回復</mark>
* <mark>最大4層まで重複可能</mark>
* <mark>恩恵共鳴が発動している場合、「免疫権限」効果は周囲の仲間にも付与される</mark>

### ★6
* 「時の魔女」状態にはいると、消費した威能ゲージの総数に応じて氷ダメージが5%アップ
* 「逆流の刻」効果が終了した際、その対象に対して期間中に与えたダメージ総量の50%の追加ダメージを与える
* ソロモードでは自身の攻撃力の10000まで、マルチモードでは自身の攻撃力の150000までの上限あり

# 恩恵目線の考察

## ガチャ目標

* 武器は5凸
* ボリションは無凸4セット
* アバター特性はコンテンツ次第なので任意(恩恵意外は使う必要なし)

## 使い方

* 戦闘の状況、他武器／ボリションの組み合わせを考慮して瞬間的に判断して仕様する必要あり、恩恵武器としては比較的使うのが難しい部類
* 連携スキル：威能ゲージ+12層、不屈、自身と仲間の会心率+16%(ボリ4)、自身と仲間の会心ダメ+7%(ボリ4)、デバフ解除、灰蝕解除
* スキル：威能ゲージ全消費、時の魔女、最終ダメージ+25%(3凸)、最終ダメージ+5%(ボリ2)、即死回避付与(5凸)
* 通常攻撃：威能ゲージを12層貯める
* 通常攻撃長押し：威能ゲージ-12層、デバフ解除、灰蝕解除、過去3.5秒間に失ったHP全回復+攻撃力*200%回復
* スキル2：威能ゲージ+12層、攻撃力*400%回復
